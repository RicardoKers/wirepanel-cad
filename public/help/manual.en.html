Wirepanel CAD User Manual

Wirepanel CAD is a browser-based 2D CAD tool for industrial electrical drawings. This manual covers drawing tools, potentials, reusable library models, intelligent components, linked parts, project files and PDF export.

Interface Overview

Top Toolbar

The toolbar uses compact icon buttons. Hover a button to see its tooltip. It contains drawing tools, view actions, project JSON actions, help and PDF export.

Center Canvas

The canvas contains the drawing sheet, grid, page frame, markers, shapes and component graphics. You can zoom, pan, select by click, select by rectangle and place components.

Left Library

The Library shows reusable models from the app and from the current project. Models are grouped by component type and each group can be collapsed.

Right Inspector

The right panel contains Object, Components, Potentials, Layers and Project sections. The Object section is organized into Component, Label and Geometry blocks.

Drawing and Editing

  • Select: select, move, resize and edit objects.
  • Line: draw regular lines.
  • Potential: draw electrical potentials with connection logic.
  • Circle and Arc: draw circular geometry.
  • Text: insert regular text, special text or linked text.
  • Pin: insert connection pins.
  • Pan: navigate the canvas without editing geometry.

Multi-selection can be done with a selection rectangle. Hold Alt while dragging to select only objects fully contained by the rectangle.

Layers

The Layers section organizes the drawing into logical levels.

  • Create, rename, show, hide, lock and delete layers.
  • Choose the active layer for newly created objects.
  • Move selected objects to another layer from the selection context menu.

Potentials

Potentials are electrical connection lines. Each potential has a number, name and wire size in mm2. Potential labels and endpoint arrows are drawn on the canvas and exported to PDF.

  • Potential endpoints can connect to pins and other potentials.
  • Connections can be created while drawing or while moving an existing endpoint.
  • The Potentials section lists the current page potentials and can renumber them.
  • Potential endpoint labels can be used as cross-reference links with Ctrl + click.

Text and Links

Regular Text

Text can be edited in the Object properties, including content, position and font size.

Linked Text

Enable the Link property and set the target address. Use Ctrl + click to navigate to the linked region.

Special Text

The following placeholders are replaced on screen and in PDF export:

  • <project>: project name
  • <drawing>: drawing name
  • <author>: author
  • <page>: current page number
  • <n_pages>: total page count

Intelligent Components

A component is a logical project element linked to one or more drawing shapes. It has an invisible componentId, an editable tag prefix, a sequential number and a visible label bound to the component.

Create a Component

  1. Draw the symbol and group its shapes.
  2. Right-click the group and choose Create component.
  3. Set the tag prefix, for example K.
  4. Set the component type, for example Contactor.
  5. Optionally choose another component in Part of.

Tags and Labels

The tag is stored as prefix plus number, such as K1 or K2. The visible label is not loose text; it is bound to the component and can be moved, aligned, resized, hidden or rotated from Object properties.

Copy and Paste

Copying a component creates a new component instance with the same characteristics and the next sequential number. The Part of relationship is cleared when copied.

Component Parts and Cross-References

A secondary component, such as an auxiliary contact, can be assigned as Part of a main component, such as a coil. Parts inherit the main component tag and number so they behave as one logical device.

  • The main component can show its parts above, below, left or right.
  • Spacing controls the distance between shown parts.
  • Offset controls the distance from the main component to the first shown part.
  • Address offset X/Y and address rotation position the reference text beside each shown part.
  • Parts are ordered by insertion order. To move a part to the end, remove Part of and add it again.
  • Use Ctrl + click on component reference text to jump to the related component or part.

Library and Reusable Models

Area Purpose Storage
App Templates Built-in reusable models bundled with the application. JSON files in src/library/components before the build.
Project Templates Reusable models available only in the current project. Inside the project JSON file.
  • Use the + button in the Library header to save the current selection as a project model.
  • Use the import button beside + to import a model JSON into the project library.
  • Click or drag a model to insert it on the canvas.
  • Right-click a project model to insert, rename, download JSON or delete it.
  • Models are grouped by component type. Groups can be collapsed to simplify navigation.
  • Exported model JSON uses the component type internally; there is no separate category prompt.

Project Files and PDF

  • Download Project: saves pages, layers, shapes, project templates and component instances to JSON.
  • Upload Project: opens a JSON file in the current application format.
  • Export PDF: exports the drawing using Project settings, including visible component labels and linked part references.

The current project format is strict. Older or partial schemas outside the current structure are rejected.

Project Settings

  • Language: switch between English and Brazilian Portuguese.
  • PDF: project name, drawing name, author, sheet, orientation and margins.
  • Grid: enable snap and set grid size and color.
  • Pins: show or hide the connection X marker.

Practical Tips

  • Use clear component types because the Library navigation is grouped by type.
  • For reusable electrical symbols, create an intelligent component before saving it as a project model.
  • Use project model JSON import/export to grow the library gradually.
  • Use Part of to connect coils, main contacts and auxiliary contacts logically.
  • Check PDF output after adjusting rotated labels or linked part references.
