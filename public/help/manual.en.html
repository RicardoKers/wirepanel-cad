2D CAD User Manual

This manual covers the main workflow of the application, including drawing tools, editing, potentials, components, project export and PDF generation.

Interface overview

Top toolbar

Contains drawing tools and global project actions.

  • Select: select, move, resize and edit objects.
  • Line: draw regular lines.
  • Potential: draw electrical potentials.
  • Circle and Arc: draw circular geometry.
  • Text: insert free or special text.
  • Pin: insert connection pins.
  • Pan: navigate the canvas without editing.
  • Fit to Page: frame the current sheet in view.
  • Download Project and Upload Project: save and open project JSON files.
  • Help: open this manual.
  • Export PDF: generate the final PDF document.

Center canvas

This is the drawing area. The sheet is displayed inside a larger viewport, allowing zoom, pan, click selection, marquee selection and snap-based placement.

Left panel

Shows available components, split between App Library and Project Components.

Right panel

Contains the Properties, Potentials, Layers and Settings tabs.

Basic drawing and editing

  1. Choose a tool in the top toolbar.
  2. Click on the canvas to start the object.
  3. Use the second click to finish lines, potentials and arcs.
  4. Switch back to Select to move, edit or multi-select objects.

Multi-selection can be done with a selection box. Hold Alt to use containment instead of simple intersection.

Layers

The Layers tab organizes the drawing into logical levels.

  • Create new layers.
  • Rename existing layers.
  • Show or hide a layer.
  • Lock a layer to prevent editing.
  • Set the active layer for newly created objects.
  • Delete extra layers when they are no longer needed.

Selected objects can also be moved to another layer through the selection context menu.

Potentials

Potentials are special lines used to represent electrical connections. Each potential has a number, name and wire size in mm².

  • While drawing, a potential can snap and connect to pins and other potentials.
  • If an endpoint touches another potential with the same number, a connection is created.
  • This also works when dragging the line or dragging one of its endpoints.
  • The Potentials tab lists potentials on the current page and provides Renumber.

Text, links and special text

Regular text

A regular text object can have free content, font size and position adjusted in the properties panel.

Linked text

Enable the Link property and fill in Target. Navigation happens with Ctrl + click on Windows/Linux or Cmd + click on macOS.

Special text

The placeholders below are replaced automatically on screen and in PDF export:

  • <project>: project name
  • <drawing>: drawing name
  • <author>: author
  • <page>: current sheet number
  • <n_pages>: total number of sheets

Components

The application works with two kinds of components.

Type Usage Where it is stored
App Library Built-in components shipped with the application. JSON files inside src/library/components before the build.
Project Components Components created inside the current drawing. Inside the project JSON file.

Create a project component

  1. Draw the elements on the canvas.
  2. Select the desired objects.
  3. Click Save to Project.

Insert a component

Click the component and then click on the canvas, or drag it to the desired position. Placement respects the grid and the symbol's internal offsets.

Generate JSON for the app library

  1. Save the symbol as a project component.
  2. Click Download JSON.
  3. Choose a file name and a category.
  4. The browser downloads the JSON file.
  5. Later, copy that file into src/library/components in the repository.
  6. Rebuild the application to include the component in the built-in library.

App library categories can be collapsed or expanded to make large libraries easier to browse.

Save, load and export

  • Download Project: creates a JSON file with pages, layers and project components.
  • Upload Project: opens a JSON file in the current application format.
  • Export PDF: generates the final document using the values from the Settings tab.

The current project format is strict. Older or partial schemas outside the current structure are rejected.

Settings

  • Language: switch between English and Brazilian Portuguese.
  • PDF: project name, drawing name, author, sheet, orientation and margins.
  • Grid: enable snap, define grid size and grid color.
  • Pins: show or hide the connection X marker.

Practical tips

  • Use Fit to Page after heavy zooming to quickly frame the sheet again.
  • Set up layers early if you plan to build large reusable component libraries.
  • For reusable symbols, create them as project components first and then download the JSON for the app library.
  • For title blocks and captions, combine regular text with the available special placeholders.
