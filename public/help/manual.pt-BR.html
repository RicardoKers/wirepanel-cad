Manual do CAD 2D para Projetos Elétricos

Este manual reúne o fluxo principal da aplicação, as ferramentas de desenho, edição, potenciais, componentes, exportação de projeto e geração de PDF.

English Manual Atalhos Componentes Textos e Links

Visão geral da interface

Barra superior

Contém as ferramentas de desenho e as ações globais do projeto.

  • Select: selecionar, mover, redimensionar e editar objetos.
  • Line: desenhar linhas comuns.
  • Potential: desenhar potenciais elétricos.
  • Circle e Arc: desenhar geometrias circulares.
  • Text: inserir textos livres ou especiais.
  • Pin: inserir pinos de conexão.
  • Pan: navegar pelo canvas sem editar.
  • Ajustar a página: enquadrar a folha no zoom atual.
  • Baixar Projeto e Carregar Projeto: salvar e abrir arquivos JSON do projeto.
  • Ajuda: abrir este manual.
  • Exportar PDF: gerar um PDF com as folhas do desenho.

Canvas central

É a área de desenho. A folha aparece dentro de uma área maior, permitindo zoom, pan, seleção por clique e arraste, além de posicionamento com snap.

Painel esquerdo

Exibe os componentes disponíveis, separados entre Biblioteca do App e Componentes do Projeto.

Painel direito

Reúne as abas Propriedades, Potenciais, Camadas e Configurações.

Desenho e edição básica

  1. Escolha uma ferramenta na barra superior.
  2. Clique no canvas para iniciar o objeto.
  3. Use o segundo clique para concluir linhas, potenciais e arcos.
  4. Volte para Select para mover, editar ou selecionar múltiplos objetos.

A seleção múltipla pode ser feita por caixa de seleção. Ao segurar Alt, a seleção passa a usar contenção em vez de simples interseção.

Camadas

A aba Camadas permite organizar o desenho em diferentes níveis lógicos.

  • Criar novas camadas.
  • Renomear camadas existentes.
  • Mostrar ou ocultar uma camada.
  • Travar uma camada para impedir edição.
  • Definir a camada ativa para novos elementos.
  • Excluir camadas adicionais quando não forem mais necessárias.

Objetos selecionados também podem ser movidos entre camadas pelo menu de contexto da seleção.

Potenciais

Potenciais são linhas especiais para representar conexões elétricas. Cada potencial possui número, nome e bitola em mm².

  • Ao desenhar, o potencial pode se conectar automaticamente a pinos e a outros potenciais.
  • Se a ponta encostar em outro potencial com o mesmo número, a conexão é criada.
  • Isso vale tanto durante o desenho quanto ao arrastar uma linha ou uma de suas pontas.
  • A aba Potenciais lista os potenciais da folha atual e permite Renumerar.

Textos, links e textos especiais

Texto normal

Um texto comum pode ter conteúdo livre, tamanho de fonte e posição ajustada no painel de propriedades.

Texto com link

Ative a propriedade Link e preencha Destino. A navegação acontece com Ctrl + clique no Windows/Linux ou Cmd + clique no macOS.

Textos especiais

Os placeholders abaixo são substituídos automaticamente na tela e no PDF:

  • <project>: nome do projeto
  • <drawing>: nome do desenho
  • <author>: autor
  • <page>: número da folha atual
  • <n_pages>: total de folhas

Componentes

A aplicação trabalha com dois tipos de componentes.

Tipo Uso Onde fica salvo
Biblioteca do App Componentes nativos distribuídos com a aplicação. Arquivos JSON em src/library/components antes do build.
Componentes do Projeto Componentes criados dentro do desenho atual. No JSON do próprio projeto.

Criar um componente do projeto

  1. Desenhe os elementos no canvas.
  2. Selecione os objetos desejados.
  3. Clique em Salvar no Projeto.

Inserir um componente

Clique no componente e depois clique no canvas, ou arraste o componente até a posição desejada. O posicionamento respeita a grade e os offsets internos do símbolo.

Gerar JSON para a biblioteca do app

  1. Salve o símbolo como componente do projeto.
  2. Clique em Baixar JSON.
  3. Escolha um nome de arquivo e uma categoria.
  4. O navegador fará o download do arquivo JSON.
  5. Depois, copie o arquivo para src/library/components no repositório.
  6. Recompile a aplicação para incluir esse componente na biblioteca nativa.

As categorias da biblioteca do app podem ser recolhidas ou expandidas para facilitar a busca.

Salvar, carregar e exportar

  • Baixar Projeto: gera um arquivo JSON com folhas, camadas e componentes do projeto.
  • Carregar Projeto: abre um arquivo JSON no formato atual do aplicativo.
  • Exportar PDF: gera o documento final usando as configurações da aba Configurações.

O formato atual do projeto é estrito. Arquivos antigos ou parciais fora do schema atual não são aceitos.

Configurações

  • Idioma: alterna entre português do Brasil e inglês.
  • PDF: nome do projeto, desenho, autor, folha, orientação e margens.
  • Grade: habilitar snap, definir tamanho e cor.
  • Pinos: mostrar ou ocultar o X de conexão.

Dicas práticas

  • Use Ajustar a página depois de um zoom intenso para voltar rapidamente ao enquadramento da folha.
  • Defina bem as camadas antes de começar bibliotecas grandes de componentes.
  • Para símbolos reutilizáveis, prefira criar como componente do projeto e depois baixar o JSON para a biblioteca do app.
  • Para carimbos e legendas, combine textos normais com placeholders especiais.
