# Basic2D CAD

Browser-based 2D CAD starter focused on electrical drawings.

## Quick start

1. Install dependencies
   npm install

2. Run dev server
   npm run dev

3. Open in browser
   http://localhost:5173

## Features in this scaffold

- Draw lines, circles, arcs, and text
- Select and move objects
- Layers with visibility/lock
- Grid and snap settings
- Save/load JSON to local file
- Export PDF (A4/A3 with scale)
- Group library (save selection, place group)

## Notes

- All data is saved locally by the user (JSON export/import)
- PDF export uses a simple unit to mm scale setting
